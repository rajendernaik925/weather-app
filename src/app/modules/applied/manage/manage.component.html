Frontend Developer

Job Overview

We are looking for a skilled Frontend Developer to join our team. You will work on creating intuitive user interfaces and provide solutions to enhance the user experience.

Responsibilities:
  • Design and implement user interface components using modern JavaScript frameworks.
  • Collaborate with backend developers to integrate APIs.
  • Ensure cross-browser compatibility and responsiveness of the application.
  • Write clean, maintainable, and scalable code.
Requirements:
  • Proficiency in HTML, CSS, JavaScript, and modern frontend frameworks (e.g., React, Angular, Vue).
  • Experience with version control tools such as Git.
  • Strong understanding of responsive design and user experience principles.
  • Excellent problem-solving skills and attention to detail.
Benefits:

  • Flexible work hours
  • Health insurance
  • Paid vacation and sick leave
